Text style transfer Evaluation Test

実験に協力して頂き、ありがとうございます。
本実験ではスタイル変換した日本語テキストについて、3項目を4点もしくは5点満点で評価して頂きます。
元テキストとモデルから生成された変換テキストの両方を読み、以下の評価を行ってください。
今回取り扱うスタイルは「フィラー・言い淀みの有無」です。
ここでの「フィラー・言い淀み」とは、発話の内容とは直接関係のない「あのー」や「えーと」のような間投詞や、
音韻の引き伸ばし、言い直しやつっかえなどを指します。
実験の手順を説明します。よく読んでからお進みください。

実験手順

  1. 名前を入力し、1から8までのセット番号を選んで下さい。
  2. "Start Experiment"を押して下さい。
  3. 元テキストと生成テキストを読み、相応しいと思った評価を選択して下さい。
    本評価実験では、生成されたテキストについて、
    「スタイル変換度合い」フィラー・言い淀みの有無が変換されているか
    「コンテンツ保存性」：元テキストの内容が保存されているか
    「自然性」：日本語として自然か否か
    の3つの観点から評価をしてもらいます。
  4. 選択を終えたら"next"ボタンを押し、次の実験に進んで下さい。
  5. 評価実験を全て終えたら"finish"ボタンを押して結果をダウンロードして下さい。
    結果は吉岡のSlackか下記のメールアドレスに送信して下さい。
  6. Emailアドレス: yoshioka.daiki@g.sp.m.is.nagoya-u.ac.jp

フィラー・言い淀みの一例

    フィラー

    「あの一」「そのー」「ま一」「ま」「えー」「え」「えと」「ん」「んと」など
    その言葉自体には意味が無い、間を詰めるためのつなぎ文句

    音韻の引き伸ばし

    「なんかー」「ですねー」など

    言い間違い

    「かうゆ通うように」「そわ備わってる」など

    言いなおし・つっかえ

    「み鏡で見んのも楽しい」「個じ個人で」など

評価時の注意

  • テキストの自然性については、言い淀みを除いた部分について考えてください
      • 例：「ああの僕はラーメンがす好きです」←フィラー・言い淀みはあるが日本語としては自然
         「僕は自動車を食べます」←日本語として不自然
         「僕はラーメンが好きですが好きですです」←言い淀みとしても不自然

その他注意

  • この実験は1セットにつき5～10分ほど要します。
  • 実験中はページの再読み込みは行わないで下さい（始めからになってしまいます）。
  • スマートフォンでは結果をダウンロードできない場合があるのでPCを使用して下さい。
